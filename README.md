# ZenAi

ZenAi is an observable, rollbackable, human-in-the-loop prompt evolution system, exploring whether language intelligence can stabilize toward a "minimal-attachment" state under continuous feedback.  
ZenAi 是一个可观测、可回滚、人工参与的提示词演化系统，用于探索语言智能是否能在持续反馈中走向“最小执念”的稳定状态。

## Project Positioning / 项目定位

- This is an experimental system; it does not claim AI has subjective consciousness or religious enlightenment  
  这是一个实验性系统，不主张 AI 具有主观觉知或宗教意义上的开悟
- Focus: whether a non-attachment language behavior pattern can be engineered  
  关注点是：是否存在可工程化逼近的非执念型语言行为模式
- Prompts are the only internal structure: versioned, rollbackable, auditable  
  提示词是系统唯一内在结构，可版本化、可回滚、可审计

## Roadmap / 路线说明

ZenAi targets public environments and follows a fully automated evolution and adjudication path.  
ZenAi 面向公共环境，采用全自动演化与裁决的公共修行系统路线。
